## How to Use This Dataset

### Option 1: Manual Entry via UI
1. Go to http://localhost:3000/submit
2. Copy each ticket description and email
3. Submit one by one
4. Watch AI triage each ticket

### Option 2: Bulk Import Script (Create this)
```javascript
// Run in browser console after logging in
const tickets = [
  {
    userInput: "My laptop won't turn on after the latest Windows update...",
    contactEmail: "john.doe@acmecorp.com"
  },
  // ... add all 25 tickets
];

async function bulkCreateTickets() {
  for (const ticket of tickets) {
    const response = await fetch('/api/tickets', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      credentials: 'include',
      body: JSON.stringify(ticket)
    });
    const data = await response.json();
    console.log(`Created ticket: ${data.ticket.id}`);
    await new Promise(resolve => setTimeout(resolve, 3000)); // Wait 3s between tickets
  }
}

bulkCreateTickets();
```

Create a seed script to insert tickets directly into the database.
